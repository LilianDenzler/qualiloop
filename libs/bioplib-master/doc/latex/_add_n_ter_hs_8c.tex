\subsection*{Usage\-: }

int Add\-N\-Ter\-Hs(\-P\-D\-B $\ast$$\ast$ppdb, B\-O\-O\-L Charmm)

This routine is used to generate a set of N-\/terminal hydrogens. By default G\-R\-O\-M\-O\-S naming is used; the Charmm flag changes this to Charmm format in which case an extra N\-T\-E\-R residue will be created. Any pre-\/existing backbone H will be removed. Typically this routine would be called after calling H\-Add\-P\-D\-B()
