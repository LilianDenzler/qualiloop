\subsection*{Description\-: }

Creates a 2\-D array where the first dimension is a set of pointers. This is better for passing into subroutines than the conventional C method of simply declaring\-:

T\-Y\-P\-E matrix\mbox{[}10\mbox{]}\mbox{[}10\mbox{]};

which, when passed to a function, loses the concept of dimensions unless the matrix is explicitly defined with these dimension in the function.

This routine creates an array of pointers to 1-\/\-D arrays and can thus be passed to functions successfully.

\subsection*{Usage\-: }

matrix = (T\-Y\-P\-E $\ast$$\ast$)Array2\-D(sizeof(\-T\-Y\-P\-E), nrows, ncolumns);


\begin{DoxyCode}
matrix = (\textcolor{keywordtype}{float} **)Array2D(\textcolor{keyword}{sizeof}(\textcolor{keywordtype}{float}), 10, 10);
\end{DoxyCode}


Returns N\-U\-L\-L (having freed any allocated memory) if there is a problem.
