\subsection*{Description\-: }

Allows code utilising Biop\-Lib to compile without modification to use function names with the 'bl' prefix.(The 'bl' prefix was introduced in July 2014.)

\subsection*{Usage\-: }

Code using deprecated functions can be compiled without modification, however, all deprecated functions use the D\-E\-P\-R\-E\-C\-A\-T\-E\-D macro which prints a warning message to stderr when a deprecated function is called.

The warning message can be suppressed as either an option at the compilation of the library or by setting an environment variable at runtime.
