\subsection*{Description\-: }

bl\-Write\-C\-S\-S\-R(fp,cssr,name,title)

This subroutine will write a C\-S\-S\-R file from a linked list of structures of type \hyperlink{structcssr__entry}{cssr\-\_\-entry}.

The strucure is set up by including the file \char`\"{}cssr.\-h\char`\"{}. For details of the structure, see this file.
