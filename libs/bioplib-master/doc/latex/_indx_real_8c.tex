\subsection*{Description\-: }

This routine uses a heapsort to index a floating point array such that arrin\mbox{[}indx\mbox{[}j\mbox{]}\mbox{]} is in ascending order with j. It is modified from the F\-O\-R\-T\-R\-A\-N version in 'Numerical Recipes' Page 233. This version correctly sorts from array element 0 as opposed to 1 in the F\-O\-R\-T\-R\-A\-N version.
