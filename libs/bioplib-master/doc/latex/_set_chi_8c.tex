\mbox{\tt in}  & {\em $\ast$next} & If N\-U\-L\-L, move all atoms in the linked list from the last atom in the torsion. Otherwise move atoms up to (but not including) next. Normally this would be the start of the next residue. \\

\hline

Sets a sidechain torsion angle in a pdb linked list. The routine assumes standard atom ordering\-: N,C\-A,C,O,s/c with standard order in the s/c.

The type input parameter is defined as follows\-:

\begin{DoxyVerb}     type     Atom names        Sequential atom numbers
     --------------------------------------------------
     0        N,  CA, CB, XG    (0 - 1 - 4 - 5)
     1        CA, CB, XG, XD    (1 - 4 - 5 - 6)
     2        CB, XG, XD, XE    (4 - 5 - 6 - 7)
     3        XG, XD, XE, XZ    (5 - 6 - 7 - 8)
\end{DoxyVerb}
