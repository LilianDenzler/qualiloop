Allocated array of strings containing chain labels. N\-U\-L\-L if unable to allocate memory.
\end{DoxyReturn}
Scans a P\-D\-B linked list for chain names. Allocates memory for an array of strings containing these labels which is returned together with the number of chains found.

N.\-B. You must free the allocated memory for both the array of chains and for each individual chain label when you've finished with it!
