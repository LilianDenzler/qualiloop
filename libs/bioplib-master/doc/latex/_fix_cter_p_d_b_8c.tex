\subsection*{Usage\-: }

B\-O\-O\-L Fix\-Cter\-P\-D\-B(\-P\-D\-B $\ast$pdb, int style)

Renames Cter oxygens in the required style (C\-T\-E\-R\-\_\-\-S\-T\-Y\-L\-E\-\_\-\-S\-T\-D, C\-T\-E\-R\-\_\-\-S\-T\-Y\-L\-E\-\_\-\-G\-R\-O\-M\-O\-S or C\-T\-E\-R\-\_\-\-S\-T\-Y\-L\-E\-\_\-\-C\-H\-A\-R\-M\-M) and generates a second oxygen if required. If the style is C\-H\-A\-R\-M\-M an extra C\-T\-E\-R residue will be created. Input may be in any of the 3 styles.
