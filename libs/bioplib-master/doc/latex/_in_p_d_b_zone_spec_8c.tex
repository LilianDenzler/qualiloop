Determines whether a P\-D\-B pointer is within a residue range specified using standard format\-: \mbox{[}c\mbox{]}nnn\mbox{[}i\mbox{]}

Also handles the residue spec of c$\ast$ (i.\-e. chain name and a $\ast$ to indicate all residues in a given chain). This must be given as resspec1 (resspec2 is then ignored).

Calls In\-P\-D\-B\-Zone() to do the actual work
