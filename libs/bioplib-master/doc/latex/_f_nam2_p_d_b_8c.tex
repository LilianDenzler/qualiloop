This routine attempts to convert a filename stem to a P\-D\-B code.

All the following inputs should produce the same output of 1fbj\-:

\begin{DoxyVerb}  $A:[PDB]PDB1FBJ.ENT
  C:\P1FBJ.PDB
  /pdb/p1fbj.pdb
  /pdb/pdb1fbj.pdb
  1fbj.pdb
  1fbjL
\end{DoxyVerb}


The routine first removes characters from the start of the filename up to the last \-: \mbox{]} / or . It then searches for the following possible patterns (where N is a digit and X is an alphanumeric)

\begin{DoxyVerb}  pdbNXXX
  pNXXX
  NXXX
  XXXX.pdb
  XXXX.ent
\end{DoxyVerb}


This should cover just about any filename which includes a legal P\-D\-B code.
