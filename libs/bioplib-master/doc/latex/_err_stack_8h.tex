\subsection*{Description\-: }

This set of routines allows a stack of errors to be created. When a program has a fatal error, the Store\-Error() routine is called to place the error on the stack. As the program un-\/winds, each routine which fails stores it's error. Finally, before the program actually exits, it calls Show\-Errors() to display the error stack.
