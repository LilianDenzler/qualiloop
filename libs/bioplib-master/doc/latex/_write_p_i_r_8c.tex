Writes a P\-I\-R sequence file from a 1-\/letter code sequence. Multiple chains are split with '$\ast$'. If By\-Chain is set the the chain\-Labels array must be non-\/\-N\-U\-L\-L and contains labels for each chain Adds a terminating $\ast$ if required.
