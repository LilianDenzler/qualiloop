\subsection*{Description\-: }

This code performs least squares fitting of two coordinate set using the method of Mc\-Lachlan as modified by Sutcliffe.

\subsection*{Usage\-: }

Passed two coordinate arrays both centred around the origin and, optionally, an array of weights, returns a rotation matrix.
