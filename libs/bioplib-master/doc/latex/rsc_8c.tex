\subsection*{Description\-: }

Takes a P\-D\-B linked list and replaces sidechains

\subsection*{Usage\-: }

The main entry point is Rep\-S\-Chain() which takes a P\-D\-B linked list, the sequence, and names of the equivalent chi table and the reference coordinate file. These files are only opened on the first call.
