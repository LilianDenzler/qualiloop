\subsection*{Description\-: }

This routine will write a .P\-D\-B file of any size from a linked list of the protein structure. This list is contained in a linked set of structures of type \hyperlink{structpdb__entry}{pdb\-\_\-entry}. The structure is set up by including the file \char`\"{}pdb.\-h\char`\"{}. For details of the structure, see this file.
